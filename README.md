# Flask Authentication App

A Flask web application that provides user authentication, password reset functionality via email, and secure configuration using environment variables.

---

## 🚀 Features

- User registration and login
- Password hashing and authentication
- Password reset via email using secure timed tokens
- Flask-Mail integration (SMTP)
- Blueprint-based project structure
- Environment variable–based configuration (`.env`)
- SQLite database with SQLAlchemy ORM

---

## 🛠️ Tech Stack

- Python
- Flask
- Flask-Login
- Flask-Mail
- Flask-WTF
- SQLAlchemy
- itsdangerous
- SQLite
